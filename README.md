# OctoHunter

This is a game development project submitted to FEI University as a coursework requirement for the Digital Game Development course (CC7140) in the Computer Science program, under the guidance of Prof. Dr. Fagner de Assis Moura Pimentel.

## Team Members
The project was developed by Guilherme Escudeiro.

## Sobre o Jogo
OctoHunter é um jogo 2D de ação no estilo Bullet Hell com elementos de cassino e deck-building de armas. Você controla um Polvo caçador de recompensas que caiu em um planeta desconhecido e precisa recuperar sua nave derrotando os nativos hostis que a roubaram.
